Технологии размещения объектов




Тег Значение
Создание фреймов
«frameset» «/frameset» Заменяет тег «body». Применяется для разделения экрана на области.
«frame» «/frame» Находится внутри тега «frameset».Внутри данного тэга находятся тэги, которые указывают на HTML-документ, предназначенный для загрузки в область.
«iframe» «/iframe» Используется для загружения документов на страницу.


Атрибут Значение
cols Используется для разбиения фрейма на колонки. Пример " cols="100,*" "
bordercolor Цвет линии границы.
frameborder Отображать рамку вокруг фрейма или нет.
name Задает уникальное имя фрейма.
noresize Определяет, можно изменять размер фрейма пользователю или нет.
scrolling Способ отображения полосы прокрутки во фрейме.
src Путь к файлу, предназначенному для загрузки во фрейме.
